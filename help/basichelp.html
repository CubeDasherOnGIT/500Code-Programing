<h1>Help For Basic 500Code Commands.</h1>

'textshow'

'Shows Text On Screen.

'exit'

Closes The 500Code Terminal.

'download'

Downloads A .FiveHundred.py File

'penmode'

Opens The 500Code Pen Mode. (AKA Graphical Interface!) Pen Mode Does Not Support Normal Mode Commands. (normal mode supported commands for pen mode: 'textshow', 'exit'.)
