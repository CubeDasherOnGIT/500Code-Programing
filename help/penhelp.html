<h1>Help For Pen Mode 500Code Commands.</h1>

Make Sure To Use The Command 'penmode' For These To Work!

'pendown'

Starts Drawing.

'penup'

Stops Drawing.

'forward'

Makes The Pen Go Forward By A Set Amount Of Pixels

'backward'

Same As 'penforward', But It Goes The Other Way.

'left'

Makes The Pen Spin Left

'right'

Same As 'penleft', But Spins Right.

'color'

Changes The Pen Color.

'pensize'

Changes The Pen Size.

'bgcolor'

Changes The Background Color.

'textshow'

Same As 'textshow' In Normal Mode, But Graphical.

'dwld'

Same As In Normal Mode.
